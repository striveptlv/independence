Cinema

A functional digital IADL activity for practicing real-world movie ticket booking skills: finding a movie, comparing showtimes, following task directions, selecting accessible or standard seats, managing ticket totals, reviewing snacks, and completing a structured mock checkout sequence with clinical support.

Choose A Booking Task

Using This With A Patient

Use the cinema task as a guided community-access simulation. The steps mirror the grocery activity: pick the goal, set the support level, let the patient work through the realistic sequence, then review choices and strategy use.

1. Choose The Booking Task

Select a task card by difficulty. Start with a familiar movie-and-time goal, or choose a harder task when the patient is ready for seat rules, budgets, or memory demands.

2. Set Support Level

Open Therapist Controls from the top navigation to adjust cues, instructions, feedback, low-vision support, contrast, and audio. Match the setup to the amount of therapist support you want to provide.

3. Preview The Task

Read the task instructions together before starting. Identify the target movie, day, time, seat needs, snack request, budget, or memory code.

4. Complete The Booking Flow

Move through movie choice, seat selection, snacks, and fake checkout one page at a time. Encourage scanning, comparison, error checking, and problem solving.

5. Review Checkout

Check that selected seats determine the ticket count, wheelchair-accessible seating is used when required, and the final mock cart matches the task directions.

6. Debrief And Document

Use the summary to discuss what worked, where cues were needed, and which skills to practice next: attention, sequencing, visual scanning, money management, or memory.
